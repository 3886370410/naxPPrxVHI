# 项目简介

## 核心功能
- 基于 YOLOv8 的桥梁损伤检测系统，可识别 5 类桥梁损伤：起泡、粉化、裂缝、剥落、锈蚀
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时检测界面，显示目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持检测结果保存和批量检测查看
- 提供完整的用户界面，可选择显示标签和原图，切换不同检测模型

## 适用场景
- 桥梁结构安全检测与评估
- 基础设施巡检自动化
- 工程质量监测
- 研究机构进行桥梁损伤分析
- 工程教学演示

## 技术特点
- 基于 Python 3.9 开发，集成 OpenCV、PyQt5、Ultralytics、PyTorch 1.9 等技术栈
- 提供完整的源代码、训练好的模型、500张标注数据集和详细运行教程
- 配置成功环境后可直接使用，支持图形化界面操作
